# Axionix Flavour Wave

```sh
cd flavour_wave
bun dev
```

```sh
cd flavour_wave_internal
bun dev
```

```sh
cd flavour_wave_internal
php artisan serve
```

## Tech Stack
- NextJS
- Laravel
- Typescript
- PHP
- Tailwindcss
- MySQL
- Laravel Backpack
- Stripe Payment Provider
- Clerk Authentication
